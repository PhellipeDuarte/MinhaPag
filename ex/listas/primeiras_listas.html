Listas de tags HTML:
  • html -> todo arquivo começa e termina com essa tag
  • head -> cabeçalho da página
  • body -> corpo da página
  • title -> título da página
  • p -> parágrafo
  • h1 a h6 -> formatação do título
  • em -> texto em itálico
  • strong -> texto em negrito
  • var -> variáveis
  • br -> quebra de linha



    Tags Básicas de todo arquivo HTML:
  1. html -> início e fim de todo aquivo
  2. head -> informações do cabeçalho da página
  3. body -> informações do corpo da página
  4. font -> define a forma da escrita

    Tags para a formatação de título:
  1. h1 -> texto XXL
  2. h2 -> texto XL
  3. h3 -> texto L
  4. h4 -> texto M
  5. h5 -> texto S
  6. h6 -> texto XS

    Tags para a formatação de texto:
  1. em -> texto em itálico
  2. strong -> texto em negrito
  3. code -> trecho de código
  4. samp -> saída de um programa
  5. kbd -> entrada do teclado
  6. var -> escrita de variáveis
